This project explores Software Defined Networking using Mininet and OpenFlow. It covers the simulation of custom network topologies and the configuration of network flows across multiple layers. Implementations include Layer 2, Layer 3, and Layer 4 forwarding, VLAN-based segmentation, multi-subnet routing, and ARP analysis. The repository demonstrates practical SDN concepts through hands-on scenarios such as simulating two-subnet topologies, configuring forwarding rules, and managing traffic behavior through programmable control.
